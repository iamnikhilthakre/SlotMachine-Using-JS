# 🎰 Lucky Slots - Slot Machine Game

A professional, modern slot machine game built with HTML, CSS, and JavaScript - perfect for learning and showcasing on GitHub/LinkedIn!

## Features
- Clean, modern UI with gradient colors and smooth animations
- Deposit, bet, and spin functionality
- 3x3 reels with 4 different symbols (A, B, C, D)
- Multiple paylines (1-3 lines)
- Real-time balance tracking
- Responsive design for mobile and desktop

## How to Play
1. Open `final.html` in your web browser
2. Enter a deposit amount and click "Deposit"
3. Choose number of lines (1-3) and bet per line
4. Click "Spin!" and watch the reels
5. If you get 3 matching symbols on a payline, you win!

## Technologies Used
- HTML5
- CSS3 (with gradients, flexbox, and media queries)
- Vanilla JavaScript (ES6+)
